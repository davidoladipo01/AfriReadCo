# 📚 AfriReadCo

AfriReadCo is a modern African book club and reading platform that celebrates African literature and builds meaningful reading communities.

Inspired by platforms like Fable and StoryGraph, AfriReadCo helps readers:

- 📖 Discover books
- 👥 Join reading clubs
- ⭐ Share reviews and ratings
- 💬 Connect with fellow readers
- 🌍 Celebrate African stories and authors

## Tech Stack

- React (Vite)
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Nodemailer
- Formik & Yup
- React Router
